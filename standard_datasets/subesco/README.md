# subesco

## Description

This directory contains a Jupyter Notebook to download the `subesco` dataset from Zenodo ([SUST Bangla Emotional Speech Corpus (SUBESCO)](https://zenodo.org/records/4526477)).


## Usage

The scripts in this directory are called automatically by `standard_datasets.py`.
